# @vitamin/cli

Vitamin 命令行入口，调用 `@vitamin/coding` SDK。

## Installation

```bash
pnpm add @vitamin/cli
```

## Usage

```bash
vitamin help
vitamin list
vitamin version
```

## Commands

- `help`：显示帮助
- `list`：列出 SDK 顶层模块
- `version`：输出 CLI 版本
